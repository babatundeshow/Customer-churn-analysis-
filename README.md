# Customer-churn-analysis-
Data Science analytics project showcasing customers who are susceptible to churn in order to enhance retention strategies and boost business growth with Python, SQL, and Power BI.
Data Science Project: End-to-End Data Analysis and Business Intelligence Solution
Overview
This project demonstrates an end-to-end data analytics workflow, transforming raw data into actionable business insights. The project includes data loading and preprocessing in Python, exploratory data analysis (EDA), database querying using SQL, interactive dashboard development in Power BI, and presentation of findings through a professional report and presentation.
The objective is to uncover meaningful patterns, identify key trends, and support data-driven decision-making.
